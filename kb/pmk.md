# Why is there a number in the installer name and what do they all mean?
## Environment
All versions of AcuCOBOL-GT extend  
All platforms  

## Situation
The installer for AcuCOBOL-GT extend on Linux contains a number 118 (setup_acucob1041pmk118shACU). What does this mean and what do all the different numbers mean?  

## Resolution
The number refers to the Product Machine Key (PMK) and relates to the OS that you are installing the product on. The PMK must be correct for the OS that it is being installed on.  
The PMK of the installed product also has to match the Machine-Key contained in the license file, otherwise there will be a machine key mismatch.
